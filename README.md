# ecommerce-customer-analytics-GOLD-APPLE

An end-to-end data wrangling, exploratory analysis, and predictive modeling project based on beauty products scraped from Golden Apple (Kazakhstan).

## Project Overview
This project investigates the global beauty and cosmetics market structure on e-commerce platforms. Using a dataset of **5,386 products** across **599 distinct brands**, we explore whether price dictates quality, analyze product lifecycles, and build machine learning models to predict product value scores and popularity levels.

## Key Findings
* **Price vs. Quality:** Price is *not* a reliable indicator of product quality. The correlation coefficient between price and rating is weakly negative ($\approx-0.1$).
* **Value for Money:** Budget and mid-range products achieve significantly higher value scores compared to luxury segments. 
* **Popularity Distribution:** Customer engagement is highly unequal and follows a long-tail distribution, dominated by a few "blockbuster" products.
* **Predictive Modeling:** Advanced machine learning frameworks like Random Forest and XGBoost achieved high accuracy (R² over 0.99) in predicting product pricing metrics and classification tasks after extensive preprocessing and handling of missing features.
